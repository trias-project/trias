# Pathways of introduction as defined by CBD

Function to get all CBD pathays of introdution at level 1
(`pathway_level1`) and level 2 (`pathway_level2`). Added pathway
`unknown` at level 1 and level 2 for classifying taxa without pathway
(at level 1 or level 2) information.

## Usage

``` r
pathways_cbd()
```

## Value

A tibble data.frame with 2 columns: `pathway_level1` and
`pathway_level2`.

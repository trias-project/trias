# Pathways of introduction as defined by CBD

**\[deprecated\]**

It is deprecated as of trias 3.1.3. Please use the data frame
[pathwayscbd](https://trias-project.github.io/trias/reference/pathwayscbd)
directly instead.

## Usage

``` r
pathways_cbd()
```

## Value

A tibble data.frame with 2 columns: `pathway_level1` and
`pathway_level2`.
